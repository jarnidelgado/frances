# Frances Trainer (starter)

Prototipo kid-friendly para practicar francés — voz fija `fr-FR`, control de velocidad, emojis y juegos (fill, drag).

## Cómo ver la app

La app está pensada como proyecto Next.js. Si la despliegas en Vercel el deploy se hará automáticamente.

## Deploy en Vercel

1. Ve a https://vercel.com y conecta tu cuenta GitHub.
2. Importa el repositorio `frances-trainer`.
3. Vercel detectará Next.js y realizará el build automáticamente.
